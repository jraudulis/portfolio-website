Projects

Face Recognition

This full-stack application is built with React on the front end, Node.js on the back end, and PostgreSQL as the database. Users can input an image URL, which triggers an API call to Clarifai’s face recognition AI model. The model processes the image, detects any human faces, and highlights them, creating an intuitive and interactive experience for visual face detection. While the database is currently offline due to a hosting plan expiration, the full front-end and back-end code are available on my GitHub profile for review.

Space Station Tracker

This project, developed with HTML, CSS, and JavaScript, tracks the real-time location of the International Space Station (ISS) on a dynamic map. The application fetches data from an API, processes the JSON response, and updates the map and marker with the received coordinates to display the ISS's current position. This was my first experience with API calls, and I found it exciting to create an interactive, real-time space-tracking application.

Gradient Color Generator

This application, built with HTML, CSS, and JavaScript, creates a customizable gradient background effect based on two user-selected colors. Users can input their desired colors, which dynamically update the page’s background with a smooth gradient transition. The chosen color codes are displayed on-screen, allowing users to easily copy and apply them in their own projects.

Simple To Do List

This is a simple, functional to-do list application built with minimal HTML, CSS, and JavaScript. Developed as a practice project, it allows users to add tasks to a list, mark them as completed, or remove them as needed. Each task is managed with intuitive buttons, making it easy to keep track of to-do items and maintain an organized workflow.
