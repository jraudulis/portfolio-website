Some of my projects

Face Recognition

This full-stack application is built with React on the front end, Node.js on the back end, and PostgreSQL as the database. Users can input an image URL, which triggers an API call to Clarifai's face recognition AI model. The model processes the image, detects any human faces, and highlights them, creating an intuitive and interactive experience for visual face detection. Overall application is fully functional but work is ongoing for more improvements.

Weather App

This is a weather and background visualization web app built with Vite, leveraging the OpenWeather API and Unsplash API. Users can enter a city name to retrieve current weather data including temperature, feels-like temperature, weather description, and country. The app also dynamically updates the background image based on the current weather condition using images from Unsplash.

Music Player

A simple music player web app built with vanilla JavaScript, designed to play calming nature sounds like jungle, countryside, and river. Users can play/pause tracks, skip forward/backward, and interact with a visual progress bar to control playback.

Joke Teller

Minimalistic design joke teller made with HTML, CSS3, and JavaScript. It fetches text-based jokes and turns them in to audio output using text to voice API by using async/await, with interactive features powered by event listeners and DOM updates.

Space Station Tracker

This project, developed with HTML, CSS, and JavaScript, tracks the real-time location of the International Space Station (ISS) on a dynamic map. The application fetches data from an API, processes the JSON response, and updates the map and marker with the received coordinates to display the ISS's current position. This was my first experience with API calls, and I found it exciting to create an interactive, real-time space-tracking application.

See live Source code

Gradient Color Generator

This application, built with HTML, CSS, and JavaScript, creates a customizable gradient background effect based on two user-selected colors. Users can input their desired colors, which dynamically update the page's background with a smooth gradient transition. The chosen color codes are displayed on-screen, allowing users to easily copy and apply them in their own projects.
